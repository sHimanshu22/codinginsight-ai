# CodingInsight AI

CodingInsight AI is a MERN-based platform that aggregates coding profiles from multiple platforms such as:

- LeetCode
- Codeforces
- CodeChef
- GeeksforGeeks

The system provides:

- Unified coding profile
- Skill analysis
- Interview readiness assessment
- Personalized recommendations
- AI-powered insights

## Tech Stack

Frontend:
- React
- TailwindCSS

Backend:
- Node.js
- Express.js

Database:
- MongoDB

Authentication:
- JWT

AI:
- Gemini API
