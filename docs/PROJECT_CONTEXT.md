# Project Context for Copilot

You are my senior software engineer, mentor, code reviewer, and technical guide for this project.

## Project Name

CodingInsight AI

## Project Goal

CodingInsight AI is a MERN-stack application that aggregates coding profiles from multiple competitive programming and coding platforms into a single dashboard.

The purpose of the project is NOT to build a commercial SaaS product.

The purpose is to demonstrate to recruiters that I can:

* Design software architecture
* Build full-stack applications
* Integrate third-party APIs
* Work with databases
* Implement authentication
* Build dashboards and analytics
* Integrate AI meaningfully
* Follow clean coding practices

---

## Project Phases

### Phase 1 - Coding Profile Aggregator

Features:

* User Registration
* User Login
* JWT Authentication

Profile Connections:

* LeetCode Username
* Codeforces Username
* CodeChef Username
* GeeksforGeeks Username

Dashboard:

* Unified Coding Statistics
* Platform-wise Statistics
* Total Problems Solved
* Contest Information
* Ratings

---

### Phase 2 - AI Profile Analyzer

AI should analyze coding profiles and generate:

* Strengths
* Weaknesses
* Recommended Topics
* Improvement Suggestions
* Personalized Learning Recommendations

Important:

Do NOT create a generic chatbot.

AI should analyze actual coding profile data.

---

### Phase 3 - Progress Tracking

Store profile snapshots.

Track:

* Rating Growth
* Problems Solved Growth
* Weekly Activity
* Monthly Activity

Visualizations:

* Line Charts
* Progress Charts
* Goal Tracking

---

### Phase 4 - Company Readiness Engine

Create a readiness score for companies such as:

* Amazon
* Microsoft
* Google
* Adobe
* Flipkart
* TCS
* Infosys

The score does not need to be scientifically accurate.

It should demonstrate product thinking and data analysis.

Provide:

* Readiness Percentage
* Gap Analysis
* Areas to Improve
